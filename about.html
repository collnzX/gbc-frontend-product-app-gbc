Project Overview

About This Project

This is a multi-tier Azure web application built to demonstrate modern cloud architecture using a frontend, backend API, and relational database.

Technologies Used

  • Azure Static Web Apps
  • Azure Functions
  • Azure SQL Database
  • GitHub for frontend deployment workflow
  • HTML, CSS, and JavaScript

Project Goals

  • Build a working cloud-hosted web application
  • Connect a real frontend to a serverless Azure backend
  • Store and retrieve structured data from Azure SQL
  • Design a scalable architecture suitable for future DR expansion

This project demonstrates practical Azure web architecture and deployment.
